## Описание

Нужно было разработать приложение для 
управления финансами.

Задача: реализовать JavaScript-логику

Вот какой функционал должен быть у сервиса:

1. **Регистрация.** Позволяет зарегистрировать нового пользователя в системе. 
Были реализованы 
структуры Sidebar.js, Modal.js, AsyncForm.js, RegisterForm.js, User.js, createRequest.js.

1. **Авторизация.** Позволяет авторизовать пользователя в системе. Были реализованы структуры Sidebar.js, Modal.js, AsyncForm.js, 
LoginForm.js, User.js, createRequest.js.

1. **Выход.** Позволяет деавторизовать пользователя в системе. Были реализованы 
структуры Sidebar.js, User.js, createRequest.js.

1. **Создание счетов.** Были реализованы структуры AccountsWidget.js, TransactionsPage.js, Modal.js, 
AsyncForm.js, CreateAccountForm.js, Entity.js, Account.js, createRequest.js.

1. **Удаление счетов.** Были реализованы структуры TransactionsPage.js, Entity.js, Account.js, createRequest.js.

2. **Создание транзакций.** Были реализованы структуры TransactionsWidget.js, Modal.js, AsyncForm.js, 
CreateTransactionForm.js, Entity.js, Transaction.js, createRequest.js.

1. **Удаление транзакций.** Были реализованы структуры TransactionsPage.js, Entity.js, Transaction.js, createRequest.js.
