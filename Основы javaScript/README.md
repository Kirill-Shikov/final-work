## Описание
Задача стояла связать API с бэкендом: чтобы данные, введённые пользователем, отправлялись на сервер, а информация, предоставленная сервером, отобразилась пользователю.



Нужно было реализовать:

1. Страница «Вход и регистрация»:
   в файле public/loginPage.js.

2. Страница «Личный кабинет пользователя»:
   в файле public/homePage.js и public/loginPage.js.
